# 二、风险导向审计流程

> **来源**：[中国注册会计师协会](https://www.cicpa.org.cn/xxfb/tzgg/202603/W020260304555799366109.pdf)

## （一）审计计划（等级 1—2）

- 初步业务活动
- 总体审计策略和具体审计计划
- 重要性

## （二）风险评估（等级 1—3）

- 风险识别和评估概述
- 风险评估程序、信息来源以及项目组内部的讨论
- 了解被审计单位及其环境和适用的财务报告编制基础
- 了解被审计单位内部控制体系各要素
- 识别和评估重大错报风险

## （三）风险应对（等级 2—3）

- 针对财务报表层次重大错报风险的总体应对措施
- 针对认定层次重大错报风险的进一步审计程序
- 控制测试
- 实质性程序

## （四）信息技术对审计的影响（等级 1—3）

- 信息技术对企业财务报告和内部控制的影响
- 信息技术一般控制、信息处理控制和公司层面信息技术控制
- 信息技术对审计过程的影响
- 计算机辅助审计技术和电子表格的运用
- 数据分析
- 不同信息技术环境下的信息管理
- 运用信息技术识别与应对舞弊风险
