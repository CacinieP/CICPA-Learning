# 四、合同法律制度

> **来源**：[中国注册会计师协会](https://www.cicpa.org.cn/xxfb/tzgg/202603/W020260304555799366109.pdf)

等级 2—3

- 合同的基本理论
- 合同的订立（要约与承诺、格式条款、免责条款、缔约过失责任）
- 合同的效力
- 合同的履行（履行规则、按份之债和连带之债、双务合同抗辩权、情势变更）
- 合同的保全（债权人代位权、债权人撤销权）
- 合同的担保（保证、定金）
- 合同的变更和转让（债权转让、债务承担、债务加入）
- 合同的权利义务终止
- 违约责任
- 主要典型合同（买卖、赠与、借款、租赁、融资租赁、承揽、建设工程、委托、运输、行纪、技术、物业服务合同）
