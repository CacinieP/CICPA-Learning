# 三、战略选择

> **来源**：[中国注册会计师协会](https://www.cicpa.org.cn/xxfb/tzgg/202603/W020260304555799366109.pdf)

## （一）总体战略（等级 1—3）

- 发展战略（一体化战略、密集型战略、多元化战略）
- 稳定战略
- 收缩战略
- 发展战略的主要途径（并购、内部发展、战略联盟）

## （二）业务单位战略（等级 1—3）

- 基本竞争战略（成本领先、差异化、集中化、战略钟）
- 中小企业竞争战略
- 蓝海战略
- 商业模式（商业模式画布、商业模式创新）

## （三）职能战略（等级 2）

- 市场营销战略、研究与开发战略、生产运营战略、采购战略、人力资源战略、财务战略

## （四）国际化经营战略（等级 2—3）

- 企业国际化经营动因
- 国际化经营的主要方式
- 全球价值链中的企业国际化经营
- 国际化经营的战略类型
- 新兴市场的企业战略
