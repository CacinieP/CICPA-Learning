# 二、战略分析

> **来源**：[中国注册会计师协会](https://www.cicpa.org.cn/xxfb/tzgg/202603/W020260304555799366109.pdf)

## （一）企业外部环境分析（等级 1—3）

- 宏观环境分析（PEST）
- 产业环境分析（产品生命周期、产业五种竞争力、成功关键因素）
- 市场环境分析（市场集中度、市场结构）
- 竞争环境分析（竞争对手分析、产业内战略群组）

## （二）企业内部环境分析（等级 1—3）

- 资源与能力分析（企业资源、企业能力、核心能力、钻石模型）
- 价值链分析
- 业务组合分析（波士顿矩阵、通用矩阵）

## （三）企业内外部环境综合分析（等级 2—3）

- SWOT 分析
