# 二、增值税法

> **来源**：[中国注册会计师协会](https://www.cicpa.org.cn/xxfb/tzgg/202603/W020260304555799366109.pdf)

等级 2—3

- 征税范围（应税交易范围界定、视同应税交易、非应税交易、进口货物、特殊项目）
- 纳税人和扣缴义务人（一般纳税人与小规模纳税人区分管理）
- 税率与征收率（13%、9%、6%、零税率、征收率）
- 一般计税方法（销项税额、进项税额、应纳税额）
- 简易计税方法
- 进口环节增值税的征收
- 出口货物和跨境销售服务、无形资产增值税政策
- 税收优惠（起征点、免征项目、即征即退等）
- 征收管理（反避税调整、纳税义务发生时间、发票管理等）
