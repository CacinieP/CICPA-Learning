# 四、企业所得税法

> **来源**：[中国注册会计师协会](https://www.cicpa.org.cn/xxfb/tzgg/202603/W020260304555799366109.pdf)

等级 2—3

- 纳税人、征税对象与税率
- 应纳税所得额（收入总额、不征税收入和免税收入、税前扣除、不得扣除项目、亏损弥补）
- 资产的税务处理（固定资产、生物资产、无形资产、长期待摊费用、存货、投资资产）
- 资产损失的所得税处理
- 企业重组的所得税处理（一般性与特殊性税务处理）
- 税收优惠（高新技术、小型微利、加计扣除、创投企业、加速折旧等）
- 应纳税额的计算
- 征收管理
