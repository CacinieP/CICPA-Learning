# 七、管理会计

> **来源**：[中国注册会计师协会](https://www.cicpa.org.cn/xxfb/tzgg/202603/W020260304555799366109.pdf)

## （一）本量利分析（等级 2—3）

- 本量利的一般关系（混合成本分解、变动成本法、基本模型）
- 保本分析
- 保利分析
- 利润敏感分析

## （二）短期经营决策（等级 1—3）

- 短期经营决策概述
- 生产决策（保留或关闭分部、自制或外购、特殊订单、约束资源、出售或深加工）
- 定价决策

## （三）全面预算（等级 1—3）

- 全面预算概述
- 全面预算的编制方法
- 营业预算的编制
- 财务预算的编制

## （四）责任会计（等级 1—3）

- 企业组织结构与责任中心划分
- 责任成本
- 成本中心、利润中心、投资中心
- 内部转移价格

## （五）业绩评价（等级 1—2）

- 财务业绩评价与非财务业绩评价
- 关键绩效指标法
- 经济增加值
- 平衡计分卡
- 绩效棱柱模型
- 标杆管理
