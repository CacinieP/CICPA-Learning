# 六、成本计算

> **来源**：[中国注册会计师协会](https://www.cicpa.org.cn/xxfb/tzgg/202603/W020260304555799366109.pdf)

## （一）产品成本计算（等级 2—3）

- 产品成本的归集和分配
- 产品成本计算的品种法
- 产品成本计算的分批法
- 产品成本计算的分步法（逐步结转、平行结转）

## （二）标准成本法（等级 1—3）

- 标准成本及其制定
- 标准成本的差异分析（变动成本差异、固定制造费用差异）

## （三）作业成本法和生命周期成本法（等级 1—3）

- 作业成本法的概念与特点
- 作业成本计算
- 作业管理
- 生命周期成本法
