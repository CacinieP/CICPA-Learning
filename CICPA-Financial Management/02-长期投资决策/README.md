# 二、长期投资决策

> **来源**：[中国注册会计师协会](https://www.cicpa.org.cn/xxfb/tzgg/202603/W020260304555799366109.pdf)

## （一）资本成本（等级 1—2）

- 资本成本的概念、应用和影响因素
- 债务资本成本的估计
- 普通股资本成本的估计
- 混合筹资资本成本的估计
- 加权平均资本成本的计算

## （二）投资项目资本预算（等级 2—3）

- 投资项目的类型和评价程序
- 投资项目的评价方法（独立项目、互斥项目、总量有限时资本分配）
- 投资项目现金流量的估计
- 投资项目折现率的估计
- 投资项目的敏感分析

## （三）期权价值评估（等级 1—3）

- 衍生工具概述
- 期权的概念、类型和投资策略
- 金融期权价值评估
- 实物期权价值评估（扩张期权、延迟期权、放弃期权）

## （四）企业价值评估（等级 2—3）

- 企业价值评估的目的和对象
- 企业价值评估方法（现金流量折现模型、相对价值评估模型）
